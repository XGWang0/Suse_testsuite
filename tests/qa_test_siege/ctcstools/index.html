qa_siege test page

This is a test page for qa_siege benchmark.

Of course, siege benchmark allows you to test any possible webpages. It even supports HTTP POSTs. As to have comparable benchmark results from different products, we created this test page, same for all of them. Before we were using http://localhost, which sometimes returns Apache welcome site, sometimes a HTTP/403, making the results incomparable.

And, yes, this page is exactly 1024 bytes long - unless someone modified it, recoded it, changed CR->CRLF, or did something similar.

If you want to know what this is all about, ask QA, vmarsik, or mmrazik.
